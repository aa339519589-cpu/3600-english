# Attributions

## Vocabulary source

The base high-school English list was adapted from:

- Repository: [shui-dun/minitools](https://github.com/shui-dun/minitools)
- Source file: [`liuji/data/gaozhong.txt`](https://github.com/shui-dun/minitools/blob/master/liuji/data/gaozhong.txt)
- License: [MIT License](https://github.com/shui-dun/minitools/blob/master/LICENSE)

Malformed source rows were corrected, duplicate headwords were merged, additional reading vocabulary was added, and usage-frequency tiers were generated locally. The application dataset contains exactly 3,600 unique entries.
